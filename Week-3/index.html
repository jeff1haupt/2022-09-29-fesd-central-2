Week 3 - Arrays, Functions, and Objects

What are arrays?

Arrays are convenient ways to store several different (but usually related) pieces of data withing JavaScript.

They can be empty or as long as you want them to be. They can contain similar types of values or differnt types of values.

This is a perfectly valid way to think of an array:

7238 3.14 1000000000 -55
0 1 2 3

As is this array:

true {name: "Jeff"} 500 "Hello World"
0 1 2 3

Just always remember you start counting the slots in an array at ZERO! And you can access the last element of an array by accessing the length method on an array like so:

someArray[someArray.length - 1]

You need the minus 1 to make up for the count starting at 0!
